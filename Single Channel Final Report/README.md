# Single Channel Final Report

This folder is intended to be copied directly into Overleaf.

## Contents

- `main.tex` - actual single-channel report draft with embedded results
- `figures/` - generated figures from the final reporting run
- `tables/` - generated LaTeX tables from the final reporting run
- `references/references.bib` - bibliography file

## Notes

- This report is based on the single-channel reporting pipeline in `reporting_pipeline/`.
- The current single-channel sweeps show that waveform quality differs substantially across models, but symbol accuracy remains saturated across the tested range.
- The report therefore argues that a harder future setting, especially the planned four-channel case, is needed to expose symbol-level failures.
