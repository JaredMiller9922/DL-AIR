This folder is the independent report package to use.

What it is:
- A standalone copy of the uploaded report structure.
- The base LaTeX file matches the uploaded main.tex everywhere except:
  - visible page numbers were added
  - the Training section was written
  - the Experiments section was written
  - the Conclusion section was written

What it does not do:
- It does not depend on the current Overleaf-style folders in this repo.
- It does not overwrite your existing report folders.

Upload guidance:
- Replace your Overleaf main.tex with this folder's main.tex.
- Upload any new figure files from this folder's figures/ directory into your existing Overleaf figures/ directory.
- Keep your existing references/references.bib unless you want to replace it with the copy in this folder.

Note:
- Some legacy figures from your current Overleaf project (such as MIT_RF_Challenge_Image.png)
  are referenced by main.tex but are not present elsewhere in this repo. Your Overleaf project already
  has them, so you do not need to re-upload those files.
