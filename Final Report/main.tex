\section{Models Used, Sources, and Modifications}
To solve the problem of RF source separation there were five different network architectures that were used. The five architectures were Linear, Hybrid, LSTM, IQ\_CNN, and HTDemucs. Below is a detailed explanation of each network.

\subsection{Linear Separator}
    The linear separator was inspired by the blind source separation technique independant component analysis. It uses a kernel\_size of $1$ and goes over the entirety of $x$. This convolution is equivalent to $y(t) = W * x(t)$. This network does well on the single channel case but doesn't learn anything meaningful in the $4$ channel case. This is as expected because once you have more than a single channel for both signals it is no longer a linear separation problem.
\subsection{Hybrid Separator}
    The hybrid separator expands on the stucture of the linear separator and adds a deep temporal model. It start by doing input projection. The idea is to learn a better representation of the mixture before we attempt to separate it. In this model we use a more standard convolution with a kernel size of 9. This allows us to learn structure over time which is critical for the multi-channel separation problem. This model acted as our baseline for the multi-channel separation problem.
\subsection{LSTM Separator}
    This network was adapted and upgraded from the Matlab LSTM network provided by Professor Tom Henderson. In this network for every example in the batch we look at the I and Q values received at each antenna. This allows the model to see the full instantaneous observation across all antennas. We then do a simple transpose to make time the sequence dimension. Without this transpose the network wouldn't be able to utilize it's memory effectively. We then do an input projection like above and pass it into an LSTM. An LSTM does the following at each time step. It takes the current input, the previous hidden state, and the previous cell state and updates it's internal memory. This allows us to get an understanding that becomes more rich over time. We use a bidrectional LSTM which allows us to learn information from before the sample and information after the sample. we also incorporate a little bit of dropout between LSTM layers to allow for better generalization.
